# Research notes (2026-07-19)

* [Steam BTD6](https://store.steampowered.com/app/960090/Bloons_TD_6/) identifies the desktop Steam app as `960090`; discovery reads the standard Steam manifest and library folders rather than relying on names.
* [MelonLoader](https://github.com/LavaGang/MelonLoader/releases) is the official upstream for release metadata. Selection is compatibility-gated, not simply newest.
* [BTD Mod Helper](https://github.com/gurrenm3/BTD-Mod-Helper) documents an in-game GitHub mod browser and is GPL-3.0. Its external browser search URL is `https://gurrenm3.github.io/BTD-Mod-Helper/mod-browser?search=`.
* GitHub Releases API is used for trusted metadata. Bytes must be hash-verified before use.
* Microsoft Store/Xbox independent-copy behavior remains unresolved, so automatic modification is disabled.
