# Profiles and mods

Profiles record mod identities, exact library files, launch arguments, configuration overrides, icon, accent, and timestamps. A launch stages only the selected profile into the managed `Mods` directory through a staging folder; disabled profiles cannot leave old user mods active.

Manual DLL/ZIP imports are retained in the central mod library and are labelled manual. They are never claimed to be verified. A ZIP must safely extract and contain exactly one DLL unless the user makes a more specific selection.
