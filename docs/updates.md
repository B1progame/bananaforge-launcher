# Updates and synchronization

Updates are independent: launcher/bootstraper releases, BTD6 source changes, MelonLoader, BTD Mod Helper, mods, and compatibility data. Compatibility remains unknown unless verified data says otherwise. BananaForge does not automatically install an incompatible update.

Before a game-source sync, back up the managed instance and preserve managed mod configuration. Use recovery journals and one-click rollback if activation fails.
