# Installation

Download `BananaForgeBootstrap.exe` only from the project GitHub Releases page. The bootstrap retrieves the release manifest from this project, constrains download origins, verifies SHA-256 before extraction, keeps the previous working version, and installs per-user by default. It never runs an unverified package.

Choose shortcuts explicitly. Uninstalling never targets an original store installation. The installer lists cached downloads, managed instances, and the mod library separately before removal.
