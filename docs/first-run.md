# First run and safety acknowledgement

The wizard records acknowledgement of the modding warning before any setup action. Choose a detected Steam/Epic/manual installation or browse to an existing desktop installation. BananaForge validates the executable and checks source/destination separation before copying.

Use a managed copy where validation and storefront behavior permit it. Microsoft Store/Xbox installs are detected but automatic modification remains disabled. The launcher does not bypass protected folders, licensing, DRM, ownership, anti-cheat, or storefront authentication.
