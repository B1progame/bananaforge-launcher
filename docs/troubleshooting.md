# Troubleshooting

Use Diagnostics to inspect source/destination safety, space, loader/helper files, duplicate DLLs, recovery journals, and redacted logs. If an operation is interrupted, choose restore or discard temporary work on next launch. Support exports redact bearer tokens, GitHub tokens, and email addresses.

For failed game launches, verify that the selected executable exists, no BTD6 process is running, the selected profile has no duplicate DLLs, and the original game was not updated since its managed copy was created.
